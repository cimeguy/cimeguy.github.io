DT-in-the-desktop

一只住在 macOS 桌面上的陶喆桌面宠物。点击换图、随机播放台词与音频，还能自己管理素材、抠图、缩放——把偶像放进 Dock 旁边陪你写代码。

→ GitHub 仓库 ← 回到作品集
🖱️

点击换图

点一下就切换陶喆的不同造型，像逗一只电子宠物。

🔊

台词 / 音频

随机播放语音台词与音频片段，桌面也能开演唱会。

🗂️

素材管理

自由导入、整理图片与音频素材，自定义你的宠物内容。

✂️

抠图 / 缩放

内置人物抠图与缩放，把任意素材处理成透明立绘。
